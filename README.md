## 文件結構
本計畫書使用 XeLaTeX 編譯，支援繁體中文。

### 目錄說明
- `frontmatter/` - 前置內容（標題頁、元資料）
- `sections/` - 各章節內容
- `main.tex` - 主文件
- `twsetup.sty` - 自訂樣式包（字型、版面設定）

## 編譯方式
```bash
xelatex main.tex
```

或使用 LaTeX 編輯器（如 TeXstudio， Overleaf）開啟 `main.tex` 並編譯。

## 字型需求
- 微軟正黑體 (Microsoft JhengHei)
- 標楷體 (DFKai-SB)
- Times New Roman
